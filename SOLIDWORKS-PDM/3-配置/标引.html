PDM标引

SOLIDWORKS PDM 可使用 Windows Search 服务来执行内容搜索。

Microsoft 不支持 Windows 10 或 Windows Server 2016 操作系统上的索引服务。 SOLIDWORKS PDM 使用 Windows 搜索服务支持在这些操作系统上执行内容搜索。
