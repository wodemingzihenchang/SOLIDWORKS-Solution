PDM自动备份

服务代理

打开SQL Server 配置管理器

双击“SQL Server 服务”列表中的“SQL Server 代理（MSSQLSERVER）”服务，启动启动该服务并将其启动模式设定为“自动”

设置维护计划

打开SQL Server Management Studio：

“管理”——“维护计划”，鼠标右键“维护计划向导”：

点击【下一步】，在“选择计划属性”对话框中将“名称”改为“自动备份”，点击右下角【更改】

如下图，将“作业计划属性-自动备份”的计划类型设置为“重复执行”并勾选“已启用”，将“频率”设置为“每天”，将“每天频率”设置为“执行一次，时间为：0:00:00”，点击【确定】：

返回“选择计划属性”对话框，点击【下一步】

如下图：在“选择维护任务”对话框中勾选“备份数据库（完整）”，点击【下一步】

如下图：在“定义备份数据库（完整）任务”对话框中，点击数据库行右侧的下拉列表箭头，在打开的数据库类型列表中，选择要备份的数据库，点击【确定】后再点击当前对话框的【下一步】：
