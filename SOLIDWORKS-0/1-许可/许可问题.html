未被授权

正版与盗版混用,一般会提示“此SOLIDWORKS版本尚未被适当地授权，请联系您的供应商”。可检查当前序列号的情况。

方法1：

1、一般会是安装盗版插件时注册表序列号被更改导致（已发现GearTax插件会出现此问题）。

2、进入注册表路径位置“计算机\HKEY_LOCAL_MACHINE\SOFTWARE\SolidWorks\Licenses\Serial Numbers”

3、将SolidWorks序列号改回正版序列号，并打开SW软件激活即可。

方法2：

若添加序列号后打开程序依旧出现弹窗，可在激活程序swactwiz.exe手动打开激活，在尝试启动SOLIDWORKS程序

方法

1.在 Windows文件资源管理器中，浏览到

  • • 您收到“独立许可证故障”或“无法获得许可证”错误。

  • • 在您转至“帮助”>“激活许可证”时没有任何反应

Q

Could not obtain license for SOLIDWORKS®.Licensed number of users reached (-4,132,0)”

Question
为什么即使 SolidNetWork License Manager 显示有可用的免费许可，仍显示以下错误消息？我收到错误信息，“Could not obtain license for SOLIDWORKS®.Licensed number of users reached (-4,132,0)”。
Answer
检查确认是否 SolidNetWork License (SNL) Manager 在运行时使用了某个 SOLIDWORKS® 选项文件，该文件本应用于为已在组中设置的用户保留 SOLIDWORKS 许可。如果用户尚未整合到 (SNL) 选项文件中包括的用户组内，则为用户生成此错误信息。要修复此错误，请关闭 SNL 管理器，编辑 SNL 选项文件，然后重新启动 SNL 管理器。

Question
是否可以将 SolidNetwork 许可证（“SNL”）或功能保留用于特定的 PC（工作站）、主机、用户或用户组？
Answer
是，可以使用“选项”文件将特定的功能保留用于特定的 PC。
\ Docs\flexuser 中的 FlexLM 帮助的 chap5.htm 已对此进行了介绍

例如，要将一个 SolidWorks 许可证保留用于名为 host_name 的 PC

1.首先创建一个名为 sw_d.opt 的文件。
2.在此文件中添加一行：
RESERVE 1 solidworks HOST host_name（其中的主机名称就是 PC 在网络上的名称。）
3.将 sw_d.opt 文件放入与 sw_d.lic 文件相同的文件夹中：..\ \licenses
4.打开 SolidNetwork 许可管理器，并通过“管理”选项卡来停止服务。当 SolidNetwork 许可管理器重新启动时，将会读取选项文件。

随后，日志文件应该会报告 sw_d.opt 文件已被读取，并且会报告任何受到影响的功能。

例如

10:30:05 (SW_D) Using options file: “C:\Program Files\2006 SNL Manager\licenses\sw_d.opt”
10:30:06 (SW_D) Server started on server1 for: solidworks
10:30:06 (SW_D) RESERVING 1 solidworks/0022 84BE 2640 DE4C license for HOST host_name

使用“选项”文件可以完成许多任务，其中之一就是排除用户/主机的功能访问权限。
