SOLIDWORKS-常见问题

无法使用剖面线

无法使用剖面线

方法

剖切线与某个面刚好相切会导致如此，例如剖面线于孔相切，导致产生厚度为0的几何体错误。

避免出现相切情况。

由于找不到mfc140u

由于找不到mfc140u.dll，无法继续执行代码。重新安装程序可能会解决此问题。

以下是解决“由于找不到 mfc140u.dll，无法继续执行代码”问题的方法：

  1. 安装 Visual C++ 运行时组件：
    • 从 Microsoft 官方网站下载并安装 Visual C++ 2015、2017 和 2019 运行时组件。这些组件是并排安装的，可以同时存在。
    • 下载链接：Microsoft Visual C++ Redistributable。
  2. 检查 SOLIDWORKS 安装包：
    • 如果问题发生在安装 SOLIDWORKS 时，确保从安装文件夹中安装所有必要的依赖项，例如 ..\SOLIDWORKS安装包\PreReqs\VCRedist 目录下的组件。
  3. 验证系统环境：
    • 确保操作系统满足 SOLIDWORKS 的最低配置要求，尤其是 Windows 更新和系统补丁是否已安装。

重命名失败

不可重命名只读文档或只读文档参考的文档。

方法：

检查文件是否真是只读文件；

检出当前打开的SW文件里，是否存在该重命名文件的上层参考装配体，且是只读的状态。

快速异常检测失败

发生了快速异常检测失败，将不会调用异常处理程序，并且进程将立即终止，

方法

关闭插件，例如PDM等和打开资源管理器窗口相关的插件，在重复操作问题验证

如果是插件影响，请卸载重装即可。

Failed to load sldresu.dll

如果您在安装 SOLIDWORKS 软件的英文版本后更改系统语言设置，则您可能会看到此错误。

SOLIDWORKS 软件会在安装路径下的语言文件夹中搜索与系统语言匹配的“sldresu.dll“文件。当 找得到该文件时，该软件将无法打开文件。

方法1：设置英文语言

可以将系统语言更改为英语

方法2：修改安装中文

打开 SOLIDWORKS 安装程序，添加所需的语言
